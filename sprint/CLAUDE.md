# The REAL Heroic Journey Sprint — Page Instructions

## What this is

This is the sales page for Christina's 4-week intensive sprint program, "The REAL Heroic Journey Sprint." It lives at go.christinarobertsenneking.com/sprint.

The page is a single HTML file (`index.html`). Everything — the words, the design, the layout — lives inside that one file.

## What Christina can ask you to change

Anything to do with the words on the page:
- Headline and subheadline text
- Program description copy
- Pricing (the investment amount)
- Dates (if a new cohort is opening)
- Testimonials (add, edit, or remove)
- The FAQ section (add new questions, update answers)
- The call-to-action button text
- Any section she feels needs updating

## What NOT to change

- The filename `Christina-Logo-nog.png` — do not rename this file or change the reference to it
- The overall color scheme (teal, gold, cream) — this is her brand
- The font choices (Cormorant Garamond and Libre Franklin)
- The overall page structure and layout — only change words unless Christina specifically asks for a design change

## How to preview changes

After making any edits, Christina can double-click `index.html` to open it in her browser and see exactly how it looks.

## How to publish changes

Once Christina is happy with the changes and GitHub is connected:

```
git add .
git commit -m "describe what you changed here"
git push
```

The page will be live at go.christinarobertsenneking.com/sprint within about 60 seconds.

## Important note for Claude

When Christina asks for changes, edit the `index.html` file directly. Always confirm what you changed and remind her to preview it in her browser before pushing live.
